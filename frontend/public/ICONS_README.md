# Icon Files

This directory contains all icon files for full PWA, favicon, and iOS home screen support. All PNG files are generated from `icon.svg` by the script below.

## Files

- `favicon.svg` - SVG favicon (modern browsers)
- `favicon.ico` - 32x32 PNG-in-ICO favicon (older browsers) - **generated by script**
- `icon.svg` - Source SVG for all generated icons
- `apple-icon.png` - 180x180 PNG for iOS Add to Home Screen - **generated by script**
- `icon-192.png`, `icon-512.png` - Backward compat (referenced by sw.js push notification) - **generated by script**
- `icons/` - Full PWA icon set: 48, 72, 96, 128, 144, 152, 192, 384, 512px plus shortcut-dashboard.png and shortcut-settings.png - **generated by script**
- `workbox/` - Local Workbox 7.3.0 modules (workbox-sw.js, routing, strategies, cacheable-response, expiration)
- `screenshots/` - Optional: dashboard.png (1280x720), mobile.png (750x1334) for install UI

## Generating Icons

```bash
npm run generate-icons
```

Or from project root: `node scripts/generate-pwa-icons.mjs` (requires `sharp` as a devDependency in frontend).

The script outputs:
- `public/icons/` - Full icon set for manifest
- `public/icon-192.png`, `public/icon-512.png` - Backward compatibility
- `public/apple-icon.png` - iOS touch icon (180x180)
- `public/favicon.ico` - ICO favicon (32x32)

The manifest (static `manifest.json` and dynamic `/api/manifest` route) and the service worker reference these icons for PWA install and push notifications.
