# Icon Files

This directory should contain the following icon files for full PWA and favicon support:

- `favicon.ico` - Multi-size ICO file (16x16, 32x32) - can be generated from favicon.svg
- `icons/` - PWA icon set (generated by script): 48, 72, 96, 128, 144, 152, 192, 384, 512px plus shortcut-dashboard.png and shortcut-settings.png
- `icon-192.png`, `icon-512.png` - Backward compatibility (generated in public/ by script)
- `apple-icon.png` - 180x180 PNG for iOS home screen
- `screenshots/` - Optional: dashboard.png (1280x720), mobile.png (750x1334) for install UI

Currently, SVG versions (`favicon.svg`, `icon.svg`) are provided as fallbacks for modern browsers.

To generate the PWA PNG icons from icon.svg:

```bash
npm run generate-icons
```

Or from project root: `node scripts/generate-pwa-icons.mjs` (requires `sharp` in frontend).

The script writes to `public/icons/` (full set for manifest) and keeps `public/icon-192.png` and `public/icon-512.png` for backward compatibility. The manifest (static and API route) and service worker use these for PWA install and push notifications.

For other formats:
1. Use an online tool like https://realfavicongenerator.net/
2. Or use ImageMagick: `convert icon.svg -resize 192x192 icon-192.png`
3. For favicon.ico, use a tool that creates multi-size ICO files
